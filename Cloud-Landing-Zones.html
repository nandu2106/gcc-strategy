Subnet Layout & IP Address Management

Standardized CIDR Allocation

Consistent IP addressing across all environments with room for growth. Each VPC allocated /16 with standardized subnet patterns per AZ.

Multi-AZ Design

Three-tier subnet design replicated across multiple Availability Zones for high availability and fault tolerance.

Three-Tier VPC Subnet Architecture
Public subnets for internet-facing resources, Private App subnets for application logic, Private Data subnets for databases with VPC endpoints for AWS services. Each tier replicated across 3 AZs.
